# Sounds Abroad — Phase 1: Infrastructure Setup

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up a Next.js 16 + TypeScript app on Vercel with Sentry observability, ready for Phase 2 data pipeline work.

**Architecture:** Next.js 16 App Router + Turbopack on Vercel. Local toolchain managed via mise (Node 24 LTS, pnpm 10). Public GitHub repo (`taewanu/sounds-abroad`) for free GitHub Actions minutes (Phase 2 needs ~6,000 min/month). Sentry free tier covers all three runtimes (client / server / edge). Domain purchase deferred to Phase 6.

**Tech Stack:** Next.js 16, TypeScript, Tailwind CSS v4, ESLint, pnpm 10, Node 24, Vercel, Sentry, GitHub.

**Note on plan format:** This is infrastructure setup, not feature work, so the strict TDD loop (failing test → implementation → passing test) does not apply. Each step still names exact files, exact commands, and expected output. Verification is via build success, dev server smoke tests, and dashboard checks.
